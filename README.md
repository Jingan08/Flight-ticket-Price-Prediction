# Flight-ticket-Price-Prediction
A Project for Machine Learning 
# Flight Ticket price prediction using machine learning ✈️
## Introduction:
- Flight Ticket Prices Matter
- Flight prices are dynamic
- Accurate prediction helps:
    - Consumer: smarter booking decisions
    - Companies: better pricing strategies and marketing
- **Goal** : Find most accurate model and suggest improvements


## Objective and Problem statement:
### This project aims to :
    1. Accurately predict flight base fare using machine learning
    2. Apply data preprocessing and cleaning
    3. Build and compare 3 regression models (LR, KNN, RF)
    4. Conduct hyperparameter tuning to optimize model performance

### Problem statement:
    1. Airfare prices are highly unpredictable due to many influencing factors.
    2. Existing models lack accuracy due to poor feature engineering or tuning.
    3. Consumer & agencies struggle to make informed decisions on ticket pricing.
    4. Airline pricing strategies are complex and hard to model.

## Model Selection and Methodology:
🚀 **Model 1 - Linear Regression (LR)** – Help us measure how much better other models
perform compared to a simple linear approach. 
 
🎯 **Model 2 - K-Nearest Neigbours (KNN)** – Useful for those dataset no clear mathematical
relationship between input features and output.Able to model local patterns in the data that linear model cannot.

🖼 **Model 3 - Random Forest (RF)** – Powerful on capturing complex, non-linear
relationship in the dataset helps us understand which features impact the fare most.  
##  Results and Analysis

The performance of each model was evaluated using R2, MAE, RMSE, and MSE:  

| Model | R2  | MAE | MSE | RMSE |
|--------|-------------|--------------|------------|-------------|
| LR |   0.6307 | 63.84 | 7899.51 |88.88 |
| KNN | 0.8341 | 20.11 | 3548.68 | 59.57 |
| RF | **0.8722** | **29.15** | **2735.10** | **52.30** |
